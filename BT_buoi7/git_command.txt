----- cách 2: tạo SSH key để kết nối github vs máy làm việc ( cty nào cũng dùng)
1️⃣ Kiểm tra máy đã có SSH key chưa (NÊN LÀM TRƯỚC)
Mở Git Bash, chạy:
ls ~/.ssh
Nếu thấy:
id_rsa
id_rsa.pub

hoặc
id_ed25519
id_ed25519.pub
👉 Đã có key → sang bước 4

Nếu:
No such file or directory
👉 Chưa có → tạo mới

2️⃣ Tạo SSH key (CHUẨN – DÙNG ED25519)
Trong Git Bash, chạy:
ssh-keygen -t ed25519 -C "your_email@gmail.com"
